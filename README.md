# Monthly Employee Assessment Automation

-> Overview
Monthly Employee Assessment Automation is a Python-based workflow automation project that streamlines the employee evaluation process.
The system automates:
* Assessment question distribution
* Employee score communication
* Acknowledgment tracking
* Excel-based performance management
Built using Python, Pandas, Gmail SMTP/IMAP, OpenPyXL, and Python-Docx.

-> Business Problem
Many organizations still manage employee assessments manually through emails and spreadsheets.
Common challenges include:
* Repetitive email communication
* Manual score distribution
* Tracking acknowledgment responses
* Managing assessment records
This project automates the entire workflow and reduces administrative effort.

-> Features
Phase 1: Assessment Distribution
* Reads questions from a Word document
* Converts questions into a structured format
* Sends personalized assessment emails to employees

Phase 2: Manual Review
* Managers review submitted assessments
* Scores and feedback are recorded in Excel

Phase 3: Score Distribution
* Reads employee scores from Excel
* Validates score data
* Sends personalized score emails

Phase 4: Acknowledgment Tracking
* Connects to Gmail via IMAP
* Tracks employee acknowledgments
* Generates response status reports
